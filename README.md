# IPL_2026_Mid_Season_Analysis

A complete end-to-end Data Analytics project based on the Indian Premier League (IPL) 2026 Mid-Season data using Google BigQuery (SQL) and Power BI.

This project focuses on transforming raw IPL datasets into meaningful insisghts through data cleaning, SQL analysis, KPI generation, and interactive dashboard visualization.

## Project Highlights
1. Cleaned and transformed raw IPL datasets using SQL in BigQuery.
2. Create analytical views and KPI queries.
3. Built an interactive Power BI dashboard with IPL themed design.
4. Performed batting, bowling, venue, and team performance analysis.
5. Analysed toss impact, match-winning patterns, and player statistics.

## Tools and Technologies 
1. Google BigQuery - Data Cleaning and SQL Anlaysis.
2. SQL - Querying and KPI generation.
3. Power BI - Dashboard and Data Visualization.

## Dataset Includes
1. Matches data
2. Ball Deliveries
3. Batting statistics
4. Bowling statistics
5. Fielding statistics
6. Venue information
7. Points Table

## SQL Objects Created
1. Cleaned Tables
2. Views
3. KPI Queries
4. Team and Venue Analysis Queries

## Dasboard Features
1. Interactive filters
2. KPI Cards
3. Team comparison Visuals
4. Toss analysis donut charts
5. Venue performance charts
6. Top player leaderboards

## Project Goals
1. SQL querying

3. Data modeling
